# <h1> SmallCV_and_SpeechRecog </h1>

SmallCV_and_SpeechRecog

This repository contains a collection of small, experimental projects focused on both computer vision and speech recognition.

It serves as a sandbox for building and testing systems that process visual input (webcam, gestures) and audio input (voice commands), with the goal of exploring how these can work together in real-time applications.

Practice core concepts in computer vision and speech recognition
Build small, modular projects for real-time interaction
Explore multimodal systems combining vision + voice

Create a foundation for more advanced interactive tools
🧪 Current Work
✋ Hand tracking using MediaPipe (computer vision)
📷 Real-time webcam processing with OpenCV
🎤 Speech recognition experiments (command-based input)
🟢 Landmark detection and coordinate mapping
🛠️ Technologies Used
Python
OpenCV (image/video processing)
MediaPipe (hand tracking)
SpeechRecognition / audio libraries (voice input)
